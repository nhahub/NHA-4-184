CPU-Only Decision: The system runs entirely on CPU. The LLM (Groq) and TTS (ElevenLabs) are external API calls. Only Whisper and the embedding model run locally, both of which perform well on CPU for short voice clips and small text queries.

7 Security Measures

MeasureImplementation
AuthenticationJWT tokens with expiration, Google OAuth 2.0 as alternative
Password StorageBcrypt hashing (never stored in plaintext)
Rate LimitingSlowAPI: 10 req/min for text, 5 req/min for voice per user
Input ValidationPydantic models validate all request bodies
Environment SecretsAll API keys stored in .env file, excluded from version control via .gitignore
CORSFastAPI CORS middleware configured for allowed frontend origins
SQL InjectionSQLAlchemy ORM with parameterised queries (no raw SQL)
OTP Verification6-digit time-limited codes for email verification during registration
