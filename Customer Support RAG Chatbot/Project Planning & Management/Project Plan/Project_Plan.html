2 Milestones & Deliverables

The project is organized around 6 core milestones. Each milestone produces specific deliverables that feed into subsequent phases.

# Milestone Timeline Deliverables
M1 Data Collection & Cleaning Week 1 – Week 3 • Raw dataset (BrownBox e-commerce Q&A — 46,924 records)
• Cleaned dataset (cleaned_data.csv)
• QA chunks dataset (qa_chunks.csv)
• Exploratory Data Analysis (EDA) notebooks
• Data cleaning & preprocessing notebooks
M2 Retrieval Week 3 – Week 5 • Sentence-Transformers embedding pipeline (all-MiniLM-L6-v2)
• ChromaDB vector database with ingested FAQ data
• Retriever module with confidence scoring (1 - distance)
• Data ingestion script (ingest_data.py)
• Embedding experiments notebook
M3 Generation Week 5 – Week 10 • LLM Generator with 4 methods (generate, rewrite, verify, direct answer)
• Adaptive RAG pipeline with 5 confidence paths
• LLM Intent Router (CASUAL vs SUPPORT classification)
• Query rewriting for vague follow-up questions
• Answer verification layer for low-confidence responses
• Pipeline benchmark test suite with MLflow logging
M4 Backend Week 4 – Week 8 • FastAPI REST API with structured endpoints
• PostgreSQL database with 5 ORM models (User, Conversation, ChatMessage, Feedback, OTPCode)
• Google OAuth 2.0 + JWT authentication + OTP password reset
• Rate limiting (SlowAPI) for critical endpoints
• Conversation history & memory management
• Voice input endpoint with Whisper speech-to-text
• Feedback system (thumbs up/down with comments)
• Structured logging middleware with unique request IDs
M5 Frontend Week 10 – Week 11 • Next.js web application
• Chat interface with conversation management
• User authentication UI (login, register, OAuth)
• Responsive design and deployment
M6 MLOps Week 8 – Week 9 • MLflow experiment tracking (logs every chat query)
• Prometheus metrics (counters, histograms, gauges)
• Streamlit monitoring dashboard with 5 KPIs and 4 charts
• Configuration comparison experiments (mlflow_experiment.py)
• /metrics endpoint for Prometheus scraping
