Project Type: Depi Graduation Project
Team Size: 5 Members
Project Duration: 11 Weeks
1 Work Distribution

The project was divided into three main parts, with team members assigned based on their expertise and areas of interest.

Part 1 Data Collection & Retrieval
Shrouk Eissa & Ahmed Elsenosy
  • Data collection & exploration
  • Data cleaning & preprocessing
  • Exploratory Data Analysis (EDA)
  • Text chunking & embedding
  • ChromaDB vector database setup
  • Retriever module & confidence scoring
Part 2 Generation & Backend
Youssef Ashraf & Ahmed Elsenosy
  • LLM Generator (Groq API)
  • Adaptive RAG pipeline
  • Intent Router (CASUAL/SUPPORT)
  • FastAPI REST API
  • PostgreSQL database & ORM
  • Authentication & security
  • Voice input (Whisper)
Part 3 Frontend & MLOps
Ashraqat Effat, Fatma Shehata & Ahmed Elsenosy
  • Next.js web application
  • Chat interface UI
  • User authentication UI
  • MLflow experiment tracking
  • Prometheus metrics
  • Streamlit monitoring dashboard
